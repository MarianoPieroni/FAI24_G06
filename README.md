# FAI24_G06
Machine Learning (ML)
Objetivo Geral
Implementar e comparar diferentes abordagens de Aprendizado de Máquina para resolver um problema específico, utilizando um ou dois conjuntos de dados públicos. O projeto abrange tanto técnicas de aprendizado supervisionado quanto não supervisionado.

Abordagens e Métodos a Serem Aplicados
1. Aprendizado Supervisionado
Aplicação de dois métodos de classificação automática para prever rótulos com base em dados históricos.

Exemplos de métodos possíveis:

Regressão Logística

Árvores de Decisão / Random Forest

SVM (Máquinas de Vetores de Suporte)

Redes Neurais Artificiais

2. Aprendizado Não Supervisionado
Aplicação dos seguintes algoritmos:

K-means: para agrupamento de dados sem rótulos prévios.

Algoritmo Apriori: para descoberta de regras de associação entre itens ou características.
